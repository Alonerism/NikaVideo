# NikaVideo (AdForge)

A token-efficient "ad factory" for generating Facebook/Instagram ad variants (copy + assets) using a Brand Kit + industry templates.

## MVP goals
- Generate many **ad variants** from a single campaign brief
- Enforce placement presets (Reels/Stories/Feed) + safe-zones
- Export a ZIP of assets + CSV manifest for upload
